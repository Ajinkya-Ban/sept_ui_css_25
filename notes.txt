Q. What is mean css selectors?
------------------------------------
The selectors is nothing but targetting element. By using the selectors we can target the html element.
The following are some the selectors which can be used in css.

a) ID selector: Id is the unique. ID selector indicated by using # symbol.
b) class selector: Class can be indicated by using dot(.). We can apply multiple classes to single element.

c) group selector: We can create group of tag as well as group of class. 
d) class with tag selector:
e) tag selector : In tag selector we can directly use html element.
f) psudeo element:
--------------------------
There are total 7 psudeo element present in css.
The psudeo element can be used for to apply style to the html element for temporary purpose.
psudeo element can not added inside the html.
The psudeo element indicated by using :: 
The following list of the psudeo elements.
The ::before and ::after can only work when we use the property 'content'.

a) ::before 
b) ::after 
c) ::first-letter
d) ::first-line
e) ::selection 
f) ::marker : ul li 
g) ::placeholder

g) psudeo classes
h) Universal selector : it is denoted by *




Q. What is mean css box model?
-----------------------------------
1. margin : Outer space of border is called as margin

margin: 10px ---> all side the margin will be 10px 
margin: 10px 20px --> 10px --> top and bottom and 20px ---> left and right
margin: 1px 5px 10px 15px 
margin:auto --> horizantally left and right side equal space adjusted.




2. padding : Inner space of border is called as padding.

padding: 10px 
padding: 10px 20px 
padding: 1px 5px 10px 15px
